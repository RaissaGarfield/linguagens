Linguagens de Programação C++


O que é


C++ é uma linguagem de programação de nível superior e de uso geral. Ela surgiu como uma extensão para a linguagem C, oferecendo funcionalidades extras, como suporte para programação orientada a objetos. Desde os anos 1990 é uma das linguagens comerciais mais populares, sendo bastante usada também na academia por seu grande desempenho e base de utilizadores.

int pot1 = 0, pot2 = 0, res = 0;
void setup()
{
  pinMode(A0, INPUT);
  Serial.begin(9600);
  pinMode(A1, INPUT);
  pinMode(12, OUTPUT);

  pot1 = 0;
  pot2 = 0;
  res = 0;
  digitalWrite(12, LOW);
}

void loop()
{
  pot1 += analogRead(A0);
  Serial.println("pot 1..............................");
  Serial.println(pot1);
  pot2 += analogRead(A1);
  Serial.println("pot 2-----------------------------------");
  Serial.println(pot2);
  if (pot1 > pot2) {
    res += (pot1 + pot2);
    Serial.println("-----------------------------------");
    Serial.println(res);
    digitalWrite(12, HIGH);
  } else {
    digitalWrite(12, LOW);
  }
  delay(1); // Wait for 1 millisecond(s)
}
