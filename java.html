Línguagem de programação Java


A programação orientada a objetos (OOP) é uma das maiores vantagens do Java. Esta é a razão pela qual essa linguagem ganhou popularidade e é apropriada para projetos de qualquer dimensão. OOP se refere à construção do seu software. Trata-se de um conceito, ou melhor, uma série de conceitos oop em Java, que possibilitam a criação de interações e relações específicas entre objetos Java para a criação e utilização eficiente de software.
Java é uma linguagem de programação orientada a objetos, multiplataforma, utilizada em bilhões de aparelhos ao redor do globo. Ele habilita aplicativos, sistemas operacionais de smartphones, softwares de negócios e diversos programas famosos.

            import java.util.Scanner;

public class programa04 {
   public static void main(String[] args) {
       Scanner sc = new Scanner(System.in);

        int pot1 = 0, pot2 = 0, res = 0;
       boolean led = false;

       System.out.print("Digite o valor do sensor potenciômetro 1: ");
       pot1 = sc.nextInt();
       System.out.print("Digite o valor do sensor potenciômetro 2: ");
       pot2 = sc.nextInt();

       if(pot1 > pot2){
           res += pot1 + pot2;
           System.out.printf(" A resposta é: %d ",res);
           led = true;
       }else{
           led = false;
       }


      sc.close();
   }
}
